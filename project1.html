13C.01.01

Working with Fonts and UI Components

Project overview

The main focus of this project was learning more about fonts and typography. The exercises helped us understand which font families are more suitable for digital platforms, what are the good font pairings and what is an appropriate text sizing and hierarchy of different text sizes. We were shown best practice in adjusting line height and padding. All of the abovementioned together with a good contrast should improve the overall look but most importantly, the readability of the website.

I chose the sans-serif font Raleway as my base paragraph font. Combined with a serif font Playfair Display, it creates an elegant yet easy-to read combination. The base paragraph text size was set to 16px. The h1-h3 headlines were adjusted accordingly with rem values following proportional text scale conventions. For example, the size of h1 was 5.653rem - following the rule of Augmented fourth.

Another challenge was designing the UI components such as buttons and cards. I chose a subtle light design to make the images stand out and be the main focal point. The second important element were the buttons.

In this exercise we were also encouraged to apply a colour palette, so I experimented with split-complementary colours: orange, purple and green. The main 2 colors were a very light grey with a purple hue and a muted purple. I chose 2 accent colors - green and orange that I used mainly for buttons and links.
